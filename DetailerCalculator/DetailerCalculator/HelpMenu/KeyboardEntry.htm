Using the Keyboard

The numberpad on the keyboard is used to enter text into the textbox on the left. Pressing enter puts that number into the output window on the right.



Pressing the enter key when the textbox on the left is empty will duplicate that last number in the stack.

If you press any of the basic operators on the numberpad on the keyboard (+, -, *, and /), then the last two numbers in the stack will be used for the calculations.
