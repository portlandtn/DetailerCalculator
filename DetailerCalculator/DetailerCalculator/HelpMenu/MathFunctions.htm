Math Functions

This calculator will perform basic math functions, just as any other calculator. The input process is very different from a standard calculator, if you're not used to using a reverse-polar notation calculator. In order for any functions to work, you have to have a number entered into the stack (enter a number and press enter). Then, you have to enter a second number and add it to the stack. After you have two numbers in the stack, you can perform the four basic math functions. The bottom number is the first number of the operation; the number above is the second.

All of the basic math functions respond to the buttons or keyboard entry on the numberpad only (Currently, functions from the main keyboard are not supported.)

Please note that text has to be entered into the stack (not the entry box) in order for the calculator to recognize the input and perform the math.
